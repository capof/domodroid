=====
Usage
=====

There is 2 usage of domodroid depending what domogik version you are using.
	
	In case you're using domogik 0.2 the Main screen is organize by room.
	
	In domogik 0.3 (current dev version) Room are now include in Domoweb UI no more in Domogik the Main screen is organize by usage.
	
In both case the map view is the same.

.. image:: map1.png

If device description is not empty it will replace the device name displayed

If you do a long click on the name of a device, it will allow you to change is description displayed. It will be lost if you sync again.
